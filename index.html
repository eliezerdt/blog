Planeación de la creacion de un modulo X

La creación del algoritmo debe de hacerse antes de ponerse a codificar, siempre solventando casos puntuales y no validaciones que pueden ya haberse hecho por el framework o entorno donde se este codificando.

Ejemplo de algoritmo (secuencia de pasos ordenados)

  1. Se asume que el usuario se encuentra autenticado, pero por motivos de ejemplo si lo validamos
  2. Verifico que no venga como parámetro un id de formulario (eso indica que es una edición)
  3. Si es Edición se cargan los campos guardados con ese id
  4. Si el formulario es nuevo colocamos un input de ejemplo 

Ejemplo de pseudocodigo (por ejemplo lo coloque en español, pero prefiero hacerlo en ingles http://en.wikipedia.org/wiki/Structured_English)
